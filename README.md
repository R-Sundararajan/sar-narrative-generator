# sar-narrative-generator
A Suspicious Activity Report Generator for Barclay's Hackathon

Setup Instructions

Follow these steps to set up and run the project locally:

1. Clone the repository
git clone https://github.com/R-Sundararajan/sar-narrative-generator.git
cd sar-narrative-generator

2. Create virtual environment
python -m venv myenv

3. Activate virtual environment
myenv\Scripts\activate

4. Install dependencies
pip install -r requirements.txt

5. Run the application
streamlit run app.py

Notes-
Do NOT commit the myenv folder.

Always activate the virtual environment before running the app.

Update requirements.txt if new dependencies are added.
